Features

Everything you need.
Nothing you don't.

CoDriver bridges the gap between AI capability and human trust — giving you full visibility and control.

🧠

Shared Browser Session

AI and human share the same live Chrome tab. No puppeteer sandbox — it's your real browser, your real accounts.

⚡

Real-Time Approval Panel

Every AI action surfaces in a side panel. One click to approve or block before anything happens. You stay in control.

🔌

MCP Protocol Native

Works with Claude, Cursor, and any MCP-compatible AI. Drop-in replacement for cloud browser tools — but local and private.

🔒

Zero Cloud, Full Privacy

Everything runs on your machine. No remote browser farms, no data exfiltration, no subscription fees. Your cookies stay yours.

🎯

Action History & Audit Log

Every click, type, and navigation logged. Review what the AI did, replay sessions, and export logs for compliance.

🚀

Instant Setup

Chrome extension + local MCP server. Install in under 2 minutes. No Docker, no VMs, no DevOps.

How It Works

Three steps to AI-human sync

1

Install Extension

Load the CoDriver Chrome extension from the GitHub release. Takes 30 seconds.

2

Start MCP Server

Run npx codriver locally. Your AI connects via the MCP protocol.

3

Approve & Go

AI takes action, you approve or block each step in real time. Full control, always.

Install

Up in 2 minutes

No Docker. No cloud accounts. Just install and go.

1

Clone and load the Chrome extension

Terminal
$

Then go to chrome://extensions, enable Developer Mode, and load the extension/ folder.

2

Start the MCP server

Terminal
$
3

Add to your AI config (Claude / Cursor)

claude_desktop_config.json
{
"mcpServers": {
"codriver": {
"command": "npx",
"args": ["codriver-mcp"]
}
}
}
